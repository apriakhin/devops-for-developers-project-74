## Requirements

- Docker (version 20.10 or higher)
- Docker Compose (version 1.29 or higher)
- Make
- Git

## Description

This project demonstrates DevOps best practices for developers, including containerization, CI/CD, and automation.

## Docker Image

The image is available on [Docker Hub](https://hub.docker.com/r/apriakhin618/devops-for-developers-project-74).

## Getting Started

### Setup the project

```bash
make setup
```

### Run the application

```bash
make dev
```

### Run tests

```bash
make test
```
